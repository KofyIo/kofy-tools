1Database IDs

Pega los IDs de cada DB en Notion. Para obtenerlos: abrí la DB en Notion, mirá la URL — el ID es la sección larga entre el último / y el ?. Se guardan en este navegador para que no los pegues de nuevo.

Sobre la DB Notificaciones: el API de Notion no permite crear DBs nuevas — solo modificar esquemas existentes. Así que primero tenés que crearla manualmente:
  1. En Notion, creá una nueva página dentro del workspace de Kofy. Llamala Notificaciones. Adentro, agregá una nueva DB (full-page).
  2. Click en los ... de la DB → Connections → agregá la integración Kofy Forms.
  3. Copiá el ID de la URL y pegalo arriba.

2Migraciones

A · Agregar Email/WhatsApp/SMS a Miembros + Clientes

Agrega 3 propiedades a Miembros (Email, WhatsApp, SMS) y 2 a Clientes (Email, WhatsApp). Idempotente — si ya existen, no pasa nada.

pendiente

B · Crear esquema de Notificaciones

Asume que ya creaste la DB Notificaciones vacía y la compartiste con la integración. Patcha el esquema completo (12 propiedades incluyendo relaciones a Órdenes, Miembros y Clientes).

pendiente

3Cargar contactos

Tiene que correr Migración A primero. Esto busca cada miembro en la DB Miembros por su nombre y actualiza Email/WhatsApp/SMS. Los números se normalizan a formato E.164 automáticamente.

pendiente

Hacelo todo de un toque

Corre Migración A → Migración B → Carga contactos, en orden.
